Pixel Scaler - A Pixel Art Upscaling Tool
=========================================

概要
----
Pixel Scaler は、ピクセルアートを高品質に拡大するツールです。  
xBR を使用し、イラストのような滑らかな仕上がりに変換します。

対応形式:
- gif / jpeg / png

使用方法
--------
1. Pixel Scaler を起動します。
2. 拡大したい画像を選択、またはドラッグ＆ドロップで追加します。
3. 拡大率（100%～10000%）を入力します。
4. 「Pixel Size」を指定します。
   - 元のピクセルアート：`1` を入力
   - すでに拡大済みの画像：その拡大倍率を入力（例: 2倍 → 2）
5. 「変換」ボタンを押します。
6. 拡大後の画像が表示されます。
7. 「出力」または「全出力」で保存できます。

ヒント
------
- オリジナルサイズのピクセルアートが最も綺麗に変換されます。
- 通常の低解像度イラスト（ピクセルアートではない画像）は綺麗に拡大できません。
- 拡大率を100%にすると、アンチエイリアス（ぼかし）処理が適用されます。
- 「Pixel Size」の設定が画像の実際のピクセルサイズと合っていないと、変換に失敗することがあります。

ライセンスと著作権
-------------------
変換後の画像は、常識的な範囲内で、個人・商用を問わず自由に使用できます。
